# Tools and code

## Mathematical model of RBC dynamics including the spleen

We have created a mathematical model of red blood cell (RBC) dynamics in a malaria infection, which is unique in including the spleen as a separate model compartment that can retain large amounts of infected and uninfected RBCs.
This model is described in our paper, [The role of the spleen in red blood cell loss caused by malaria: a mathematical model](https://doi.org/10.1371/journal.pcbi.1013865) (PLOS Comp Biol 22(1), 2026), and is freely available as an R package with online documentation:

[The `spleenrbc` R package](https://rgmoss.pages.gitlab.unimelb.edu.au/malaria-spleen-rbc-loss/){ .md-button .md-button--primary }
